# MakeyMakey של טל

A webapp of fun games and activities for [MakeyMakey](https://makeymakey.com/), designed for Tal (age 9).

## Games

| Game | Keys |
|------|------|
| 🥁 Drum Machine | ↑ ↓ ← → Space Enter |
| 👾 Pac-Man | ↑ ↓ ← → |

## Running

Just open `index.html` in a browser — no build step needed. Works on Chromebook.

## MakeyMakey Setup

Plug in your MakeyMakey, connect alligator clips to the arrow pads and space bar, then touch the clips while grounded.
